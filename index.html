aprendendo CSS

o que é Css?

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo utilizada para definir a aparência e o layout de documentos escritos em linguagens de marcação, sendo a mais comum o HTML. Enquanto o HTML é responsável pela estrutura (como o "esqueleto" de um site), o CSS cuida da parte visual: cores, fontes, espaçamentos, animações e a disposição dos elementos na tela.



Funcionalidade Principais

  • Layout: Controle de posicionamento usando Flexbox e Grid.
  • Responsividade: Uso de media queries para adaptar o layout a diferentes tamanhos de tela.
  • Estilização: Definição de cores, fontes, bordas, espaçamentos (padding/margin) e tamanhos.
